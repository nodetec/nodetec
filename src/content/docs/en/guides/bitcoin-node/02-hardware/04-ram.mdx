# Random Access Memory (RAM)

Here are the current low-end, mid-range, and high-end recommendations for the node's Random Access Memory (RAM).

## Size

Current size recommendations:

- Low-end: 2 GB

- Mid-range: 4 GB

- High-end: 8+ GB

## Type

Currently, **DDR4 SDRAM** is recommended for low-end, mid-range, and high-end nodes.
